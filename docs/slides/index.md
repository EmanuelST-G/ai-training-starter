# Folien – AI-Schulungspaket

Übersicht aller 18 Reveal.js-Folien-Decks, gruppiert nach Lernphase. Jedes Deck ist eine eigenständige HTML-Datei in einem der Verzeichnisse `phase-*/`.

!!! info "Hinweis"
    Die unten gelisteten HTML-Dateien entstehen schrittweise in Plan D Tasks 2–4. Bis dahin sind die Links als Pfad-Angaben (`phase-X-y/01-z.html`) und kurze Inhaltsangaben sichtbar; der Build bleibt strict-clean. Sobald die Decks vorliegen, werden aus den Pfad-Angaben echte Markdown-Links.
